
Các loại SQLi
 + In-Band : Dùng để trực tiếp truy xuất để xem thông tin trong csdl
 + Error-Based : Tạo ra lỗi để biết các thông tin về phiên bản SQL
 + Union-Based: Sử dụng lệnh `UNION` để trích xuất thêm các dữ liệu trong SQL
